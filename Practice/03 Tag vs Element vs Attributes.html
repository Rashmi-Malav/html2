Tag vs Element vs Attributes

Tag

Tag are those which comes in between < angular brackets>

90% tags are come with starting and closing tags. eg. h1, p, head, html, body etc

10% tags are come with self closing tag or void tag or empty tag. eg. hr, br, img, input, meta etc.

Elements

Start tag - some content - End tag

Attributes

They provide some additional information which change the behaviour of element. Always come in start tag
